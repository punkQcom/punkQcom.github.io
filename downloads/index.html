Installation — macOS

  1. Extract the downloaded zip file
  2. Copy vstcollaboration.vst3 to ~/Library/Audio/Plug-Ins/VST3/
  3. Remove the quarantine flag:
    xattr -cr ~/Library/Audio/Plug-Ins/VST3/vstcollaboration.vst3
  4. Restart your DAW

Installation — Windows

  1. Extract the downloaded zip file
  2. Copy vstcollaboration.vst3 to C:\Program Files\Common Files\VST3\
  3. Restart your DAW

System Requirements

macOS: Apple Silicon (arm64), macOS 11+

Windows: x86_64, Windows 10+


Supported DAWs

Any DAW with VST3 support — Ableton Live, FL Studio, Logic Pro, Reaper, Cubase, Studio One, Bitwig, and more.
