Installation — macOS

Your browser may show "Suspicious download blocked" for the macOS zip. This is normal for unsigned plugins — click the warning and choose "Keep" or "Download suspicious file" to proceed.

The zip contains a vstcollaboration.vst3 folder:

vstcollaboration.vst3/
├── Contents/
│   ├── MacOS/
│   │   └── vstcollaboration
│   └── Info.plist
  1. Extract the downloaded zip file
  2. Copy the entire vstcollaboration.vst3 folder to ~/Library/Audio/Plug-Ins/VST3/
  3. Remove the quarantine flag (required — the plugin will not load without this):
    xattr -cr ~/Library/Audio/Plug-Ins/VST3/vstcollaboration.vst3
  4. Restart your DAW

Installation — Windows

The zip contains a vstcollaboration.vst3 folder:

vstcollaboration.vst3/
├── Contents/
│   └── x86_64-win/
│       └── vstcollaboration.vst3
  1. Extract the downloaded zip file
  2. Copy the entire vstcollaboration.vst3 folder to C:\Program Files\Common Files\VST3\
  3. Restart your DAW
