Transparency Report

Last updated: March 24, 2026

Warrant Canary

As of the date above:

  • punkQ has not received any National Security Letters or FISA court orders.
  • punkQ has not received any government requests for user data.
  • punkQ has not received any court orders requiring disclosure of user data or encryption keys.
  • punkQ has not been required to modify its systems to allow third-party access to user data.
  • punkQ has not placed any backdoors in its software or infrastructure.

How We Handle Data Requests

If we ever receive a valid legal request for user data, we will:

  • Verify that the request is legally valid and specific
  • Narrow the scope of disclosure to the minimum required by law
  • Notify the affected user unless legally prohibited from doing so
  • Log all access in our append-only audit trail

Data We Hold

For full details on what data we collect and store, see our Privacy Policy. In summary:

  • Account data: name, email, OAuth provider ID
  • Files: user-uploaded files stored in Azure Blob Storage, encrypted at rest by Azure
  • Metadata: file names, sizes, timestamps, project membership, activity logs
  • No passwords: we use OAuth exclusively and never store passwords
  • No payment details: handled entirely by our third-party payment provider

Update Schedule

This page is reviewed and updated at least once per quarter. If this page is not updated for more than 90 days, or if any of the above statements are removed, users should assume the corresponding statement is no longer true.
