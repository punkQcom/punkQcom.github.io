Frequently Asked Questions


What is VST Collaboration?

VST Collaboration is a VST3 plugin that runs inside your DAW (Ableton Live, FL Studio, Reaper, etc.). It lets you upload, download, and manage project files directly from within your session — no need to switch to a browser or file manager. Sign in with Google, GitHub, Microsoft, or Discord to get started.

What platforms and DAWs are supported?

The plugin is available for macOS (Apple Silicon) and Windows (x86_64) as a VST3 plugin. It works in any DAW that supports VST3 — confirmed working in Ableton Live on both platforms. Other DAWs (FL Studio, Logic Pro, Reaper, Studio One, Cubase) are planned for testing.

Where are my files stored?

Files are stored in Microsoft Azure Blob Storage across three regions: Europe (Ireland), United States (Virginia), and Asia (Singapore). Your storage region is automatically assigned based on your location at first login, so your files are always stored close to you for the fastest uploads and downloads.

Is my data secure?

Yes. All files are encrypted at rest by Azure Storage. Access is controlled via time-limited, blob-level security tokens issued by the backend only after OAuth authentication. The plugin never stores storage account keys. All communication uses HTTPS/TLS. OAuth sign-in uses PKCE for maximum security. End-to-end (zero-knowledge) encryption is on the roadmap.

How does collaboration work?

Create a Project, then invite collaborators by email. Each member gets a role: Owner (full control), Editor (upload, rename, delete), or Viewer (download only). Project storage counts toward the owner's quota, not the collaborators'. Drag and drop files between your Personal storage and any Project folder.

What is QuickShare?

QuickShare lets you create a temporary web link to share a folder with anyone — no account needed. Recipients can browse and download files in their browser. You can choose Read Only or Read & Write access. Links expire automatically (15 minutes on Free, 60 minutes on Starter, 2 hours on Pro). You can also send the link directly via email from within the plugin.

What file types can I upload?

Audio, video, music project files, samples, presets, documents — virtually anything music production related. Executables, disk images, and system files are blocked for security. There is no file size limit beyond your storage quota.

What do I get on the Free plan?

200 MB of storage, 1 project with 1 collaborator, and 1 active QuickShare link (15-minute expiry). No credit card required — just sign in with any OAuth provider and verify your email to activate your storage.

Can I use multiple sign-in providers?

Yes. If you sign in with different providers (e.g. Google and GitHub) that share the same verified email address, your accounts are automatically linked. You'll see the same files regardless of which provider you use to sign in.

How do I delete my account?

Go to the Account tab in the plugin and click "Delete Account". This permanently removes all your files, project data, and database records. The action cannot be undone.
