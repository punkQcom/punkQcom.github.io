Frequently Asked Questions

How to use the plugin, plus answers to common questions

Usage Guide


How do I upload files?

Drag files from Finder (macOS) or Explorer (Windows) directly into the plugin window. You can also right-click any folder and choose Upload to browse for files. Uploads go to whichever folder you drop them on or have selected.

How do I download files to my DAW?

Drag any file from the plugin into your DAW's timeline or arrangement view. The file is downloaded and imported in one step. Downloaded files are cached locally, so dragging the same file again is instant.

How do I organize my files?

Right-click any file or folder for options: rename, move, copy, delete, or create a new folder. You can also drag files between folders, and between your Personal storage and Project folders.

How do I share files with collaborators?

Create a Project from the Projects tab, then invite collaborators by email. Each member gets a role: Owner (full control), Editor (upload, rename, delete), or Viewer (download only). All project members see the same files in real time.

How do I share files with anyone (no account needed)?

Right-click any folder and choose QuickShare to create a temporary web link. Anyone with the link can browse and download files in their browser. You can copy the link or send it via email directly from the plugin. Links expire automatically. Note: QuickShare files are not encrypted, since recipients access them via web browser without an account.

How do I preview audio?

Click the play icon on any audio file in the file tree. Use this to quickly audition files and compare different versions without importing them into your DAW.

How do I use version history and comments?

Click the file icon or right-click a file and choose History & Details. You can see all previous versions of a file, restore an older version, and leave comments for collaborators.

What about encryption?

Files in your Personal storage and Project folders are encrypted automatically on your device before upload -- no action needed from you. QuickShare files are not encrypted, since recipients access them via a web link without an account.

Where are cached files stored?

The plugin caches downloaded files locally so you can re-use them without downloading again. You can manage cache size and automatic cleanup in the Options tab, or clear the cache at any time.

How do I change language or theme?

Open the Options tab to choose from 9 available languages. Use the theme toggle in the title bar to switch between light and dark mode.


General


What is VST Collaboration?

VST Collaboration is a VST3 plugin that runs inside your DAW (Ableton Live, FL Studio, Reaper, etc.). It lets you upload, download, and manage project files directly from within your session — no need to switch to a browser or file manager. Sign in with Google, GitHub, Microsoft, or Discord to get started.

What platforms and DAWs are supported?

The plugin is available for macOS (Apple Silicon) and Windows (x86_64) as a VST3 plugin. It works in any DAW that supports VST3 — confirmed working in Ableton Live on both platforms. Other DAWs (FL Studio, Logic Pro, Reaper, Studio One, Cubase) are planned for testing.

What file types can I upload?

Audio, video, music project files, samples, presets, documents — virtually anything music production related. Executables, disk images, and system files are blocked for security. Uploads from the plugin have no per-file limit beyond your storage quota. QuickShare web uploads have a per-file limit based on your plan: 50 MB (Free), 250 MB (Starter), or 500 MB (Pro).

Can I drag and drop files?

Yes. Drag files from Finder or Explorer into the plugin to upload. Drag files from the plugin into your DAW to download and import. You can also drag files between folders, and between Personal storage and Project folders.


Storage & Security


Where are my files stored?

Files are stored in secure cloud storage across three global regions (Europe, United States, and Asia). Your region is automatically assigned based on your location at first login, so your files are always close to you for the fastest uploads and downloads.

Is my data secure?

Yes. Files in your Personal storage and Project folders are encrypted on your device before upload and decrypted only when you download them. All communication is encrypted in transit, and access is controlled via short-lived security tokens. QuickShare files are not encrypted, since recipients access them via a web link without an account.

Can I use multiple sign-in providers?

Yes. If you sign in with different providers (e.g. Google and GitHub) that share the same verified email address, your accounts are automatically linked. You'll see the same files regardless of which provider you use to sign in.


Collaboration


How does collaboration work?

Create a Project, then invite collaborators by email. Each member gets a role: Owner (full control), Editor (upload, rename, delete), or Viewer (download only). Project storage counts toward the owner's quota, not the collaborators'. Drag and drop files between your Personal storage and any Project folder.

What is QuickShare?

QuickShare lets you create a temporary web link to share a folder with anyone — no account needed. Recipients can browse and download files in their browser. You can choose Read Only or Read & Write access. Links expire automatically (15 minutes on Free, 60 minutes on Starter, 2 hours on Pro). You can also send the link directly via email from within the plugin. Note: QuickShare files are not encrypted, since recipients access them in a browser without logging in.

Who pays for project storage?

The project owner. All files uploaded to a project count toward the owner's storage quota. Collaborators (Editors and Viewers) do not need storage of their own to participate in a project.


Plans & Account


What do I get on the Free plan?

The Free plan is fully functional — every feature works, including file uploads, audio preview, comments, version history, projects, and QuickShare. You get 200 MB of storage, 1 project with 1 collaborator, and 1 active QuickShare link (15-minute expiry). No credit card required — just sign in and verify your email to get started. Paid plans add more storage, projects, and higher limits.

How do I upgrade to a paid plan?

Open the Account tab in the plugin and choose Starter or Pro. Subscriptions are billed monthly in EUR via LemonSqueezy. You can upgrade, downgrade, or cancel at any time from the Account tab.

How do I delete my account?

Go to the Account tab in the plugin and click "Delete Account". This permanently removes all your files from cloud storage, all project data, and all database records. The action cannot be undone.
